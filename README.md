# Машинное обучение — НИУ ВШЭ
 
Решения домашних заданий курса «Интеллектуальный анализ данных».
Стек: Python, numpy, pandas, scikit-learn, XGBoost / CatBoost / LightGBM, nltk.
 
## Содержание
 
| № | Тема | Описание |
|---|------|----------|
| 01 | numpy | Векторизованные вычисления без циклов. |
| 02 | pandas | Практика основных методов pandas: groupby, pivot, melt и другие. |
| 03 | Разведочный анализ (EDA) | Исследование данных об активности людей: по визуализациям подбираются пороги, разделяющие пользователей на классы. |
| 04 | kNN и линейные модели | kNN (визуализация решающих поверхностей и собственная реализация) и линейные модели: регрессия, регуляризация Ridge/Lasso, работа с признаками. |
| 05 | Градиентный спуск | Реализация градиентного спуска с нуля на собственных классах. |
| 06 | Классификация текстов | Токенизация, векторизация (Count / TF-IDF), стемминг, логистическая регрессия, работа с частотами признаков. |
| 07 | Деревья и случайный лес | Реализация решающего дерева с нуля, бэггинг и собственный случайный лес. |
| 08 | Бустинг и кластеризация | Часть 1 — сравнение XGBoost / CatBoost / LightGBM с линейным baseline.<br>Часть 2 — кластеризация: сегментация сборников музыки по общим слушателям, с подбором гиперпараметров и визуализацией. |
